# Product Overview

Bagger Flow is a VS Code color theme extension providing multiple dark theme variants. It ships as a `.vsix` package published to both the VS Code Marketplace and Open VSX Registry.

## Theme Variants

- **Dark Navy (default)** — the canonical theme, mirrored from `bagger-flow-dark-classic.json`
- **Dark Gray** — neutral zinc-toned variant
- **Dark Cherry** — warm rose/red-toned variant
- **Dark Teal** — cool teal-toned variant

All variants are generated programmatically from the single classic base theme by swapping a defined color palette.

## Target Languages

Primary syntax highlighting support: HTML, CSS, SCSS, JavaScript, TypeScript, JSX/TSX, Ruby, and Liquid templates.
